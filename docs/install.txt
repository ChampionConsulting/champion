NOTES:

Some computers hide files that start with "." so the .htaccess file may not have been uploaded when you installed. In this case upload the sample.htaccess then rename to just .htaccess on your server.

If you have trouble saving content or settings, check permissions on the content folder and the config.php file. As a general rule, permissions for folders should be set as 755 and files 644. Running yoursite.com/admin/install.php will help with this.
